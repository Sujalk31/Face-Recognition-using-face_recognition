# 🎯 Face Recognition System (Enrollment + Video Recognition)

A simple yet powerful face recognition project that allows you to:

✅ Enroll faces using a webcam  
✅ Build a face database automatically  
✅ Recognize known people in videos  
✅ Label faces with confidence scores  
✅ Save processed output video  

Built using Python, OpenCV, and the face_recognition library.

---

## 📌 Features

- Real-time face enrollment via webcam
- Stores face encodings in a compressed database
- Recognizes faces in video files
- Displays name + confidence score
- Handles unknown faces
- Efficient processing (frame skipping + scaling)
- Easy to use command-line interface
